# Password Generator

A simple password generator made using HTML, CSS, and JavaScript.  
It includes a show/hide password option using an eye icon and works well on both desktop and mobile screens.

## Features
- Show / hide password
- Responsive design
- Clean and simple UI

## Technologies
- HTML
- CSS
- JavaScript

## Usage
Open `index.html` in your browser and use the eye icon to show or hide the password.
